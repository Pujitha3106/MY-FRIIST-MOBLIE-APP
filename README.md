# SOS Emergency call app(Kodular)
The *SOS Emergency Call App* is a simple and fast emergency application bulit using *Kodular* with a singe app instantly makes a call to your saved emergency contact.This app*does not use location , GPS , or internet- it focuses only on *emergency calling*
# Features
1.One-Tap Emergency call-instantly calls the emergency number you set.
2.Fast & LightWeight-Works without internet,no location or SMS features.
3.Simple UI-Big SOS button for quick access.
4.Privacy Friendly-No tracking,No data collection,No permission except CALL.
# Built with
*kodular creater*
Components used:
1.Call phone component
2.Button componet
3.Layout&Label Components
# How It Works
1.User opens the apps
2.User should enter the number.
3.Presses the *SOS button*
4.The app immediately starts a phone call to the emergency number.
# INSTALLATION
1.clone the repository bash git clone-https://github.com/Pujitha3106/MY-FRIIST-MOBLIE-APP/tree/main
2.Import the .aia file into *Kadular Creater*
3.Edit the emergency number.
4.Export the app as APK.
5.Install on your mobile device.
