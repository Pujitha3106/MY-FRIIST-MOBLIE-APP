# SOS Emergency Call App
The *Sos Emergency call app* is a simple and fast emergency application built using *Kodular* with a single tap, the app instantly makes a call to your saved emergency contact.
This app *does not use location, GPS, or internet*-it focuses only on *emergency calling.

# Featuers 
# One - Tap Emergency Call
Instantly calls the emergency number you set.
# Fast & Lightweight
1.Works without internet.
2.No location or SMS features.
# Simple UI
Big SOS button for quick access.
# Privacy Friendly
1. No tracking.
2. No data collection.
3. No permission expect CALL.
# Bulit With
*Kodular Creater*
Componets used:
1.Call Phone. 
2.Button.
3.Layout&Label.
# How it works
1.User opens the app
2.Presses the *SOS button*.
3.Enter the number.
4.The appa immedeately starts a phone call to the emergency number.
# Installation 
1.Clone this repository 
2.Import the .aia file into *Kodular Creater*.
3.Edit the emergency number.
4.Export the app as APK.
5.Installation on your phone.
